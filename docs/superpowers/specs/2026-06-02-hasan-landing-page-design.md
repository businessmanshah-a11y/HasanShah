## ۳. تکنولوژی

- **Stack:** HTML5 + CSS3 + Vanilla JavaScript (بدون فریم‌ورک)
- **انیمیشن:** GSAP (ScrollTrigger، Timeline) + Particles.js یا Three.js برای Hero
- **فرم backend:** Google Apps Script (Google Sheets) + Telegram Bot API
- **هاست:** GitHub Pages (repo: businessmanshah-a11y/HasanShah)
- **ساختار فایل:**
  ```
  index.html
  assets/
    css/
      style.css
      animations.css
    js/
      main.js
      form.js
      particles.js
    fonts/
    images/
      portfolio/
      profile/
  ```

## ۶. یکپارچه‌سازی فرم

### Google Sheets
- ارسال با Google Apps Script (fetch POST به Web App URL)
- هر ثبت‌نام یک ردیف: timestamp، نام، تلگرام، نوع کسب‌وکار، همه پاسخ‌ها
- آپلود لوگو: فایل در frontend به base64 تبدیل می‌شه و به Apps Script ارسال میشه — Apps Script فایل رو در Google Drive ذخیره و لینک رو در شیت ثبت می‌کنه

### Telegram Bot
- پیام فوری بعد از ثبت:
  ```
  🔔 لید جدید!
  👤 نام: ...
  📱 تلگرام: ...
  🏢 کسب‌وکار: ...
  🎨 رنگ: ...
  📊 هدف: ...
  ```
