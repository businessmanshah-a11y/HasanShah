# HasanShah Landing Page

Professional bilingual landing page for Hasan Shahmoradi.

## Deploy

Push to GitHub and enable Pages from repo Settings → Pages → main branch.

## Before Going Live

1. Add your Google Apps Script Web App URL in `assets/js/submit.js` (replace `YOUR_WEB_APP_URL_HERE`)
2. Deploy `google-apps-script/Code.gs` — see `google-apps-script/SETUP.md`
3. Replace portfolio placeholders in `assets/images/portfolio/` with real screenshots
4. Push: `git push -u origin main`
